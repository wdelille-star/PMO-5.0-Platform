# PMO-5.0-Platform

Digital Maturity Self-Assessment SaaS for Organizations

PMO-5.0-Platform is a web-based self-assessment platform designed to evaluate the digital maturity level of organizations. The system allows companies to complete an interactive questionnaire and instantly receive a scored evaluation, maturity level, and decision-support insights.

This project is inspired by Industry 5.0 and Business Intelligence concepts, focusing on automation, data-driven evaluation, and user-friendly dashboards.

Features
Interactive digital maturity questionnaire
Automated scoring engine
Maturity level classification (Beginner / Intermediate / Advanced)
Company response storage (SQLite database)
Admin dashboard to view all assessments
Clean UI for quick evaluation
SaaS-ready architecture
Backend API using Python Flask
